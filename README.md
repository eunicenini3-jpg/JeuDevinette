L'ordinateur génère un nombre aléatoire compris entre 1 et 100. puis l'utilisateur doit essayer de retrouver ce nombre. L'utilisateur dispose de 10 tentatives maximum. Après chaque proposition : Si le nombre saisi est inférieur au nombre recherché → le programme indique "Nombre proposé plus grand", Si le nombre saisi est supérieur au nombre recherché → le programme indique **"Nombre proposé plus petit", si l'utilisateur trouve le bon nombre → un message de victoire est affiché, si les 10 tentatives sont utilisées sans succès, un message de défaite apparaît.
